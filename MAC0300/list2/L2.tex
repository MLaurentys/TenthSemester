	\textbf{6.4}\\
	\noindent If using float-point precision:\\
	
	$1 \times 2^{-24} = $ \floatRepr{0}{01100111}{0}
	
	\noindent If using double-point precision:\\
	
	$1 \times 2^{-53} = $ \doubleRepr{0}{01111001010}{0}\\
	
	\noindent
	\textbf{6.10}\\
	Correctly rounded:\\
	
	Result $=$ \floatRepr{0}{01111111}{1}\\
	
	One guard bits:\\
	Result $= 1$. One guard bit is not enough as, after shifting 24 places to the right, the second term would be filled with zeros with a single one on the guard bit. If rounding up, the result would be correct.\\
	
	Two guard bits:\\
	Result $= 1$. Again, two guard bits are not enough as, after shifting 24 places to the right, the second term would be filled with zeros with a single one on the first guard bit and another zero on the second guard bit. If rounding up, the result would be correct.\\
	
	Two guard + 1 sticky bits:\\
	Result $=$ \floatRepr{0}{01111111}{1}. The number will be correctly rounded.
